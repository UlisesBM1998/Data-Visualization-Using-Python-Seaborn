📈 I developed a set of data visualization projects in Python using Seaborn, each one based on a real business problem.

📈 The objective of this project is to transform raw data into clear visual insights that support better decision-making across different business areas. Each visualization answers a specific business question, focusing not only on charts, but on how to communicate patterns, trends, and anomalies effectively.

📈 The project includes 5 real-world scenarios such as:

◉ Customer Analytics (Histogram)
◉ Sales Performance (Barplot)
◉ Marketing ROI (Heatmap and Scatterplot)
◉ Human Resources (Boxplot, Swarmplot, Violinplot)
◉ Financial Analysis (Lineplot)
